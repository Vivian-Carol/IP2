UNEMPLOYMENT INDEX IN THE WAKE OF COVID-19 RECOVERY

Page Content

The dataset represents the number of unemployed people in thousands, recorded across approximately 60 countries worldwide for the years 2019, 2020, and 2021. The data provides insights into the performance of the global labor market before, during, and after the COVID-19 pandemic and related restrictions. By analyzing the trends observed in this dataset, valuable inferences can be drawn about how the labor market might respond to potential future disasters or crises. Understanding the pre-COVID, during-COVID, and post-COVID labor market conditions enables policymakers, economists, and researchers to make informed decisions and implement strategies to support and strengthen the workforce in times of uncertainty and recovery

Resources

To fetch the current data on unemployment rates from the OECD API, the application uses a JavaScript function that makes an asynchronous API call to the OECD data server. The response is in XML format, and the application then parses the XML data to extract the relevant unemployment rate information for different countries and years. The data is then processed and displayed in a tabular format and visualized in an interactive line chart using the Chart.js library.
